# shex.c
A Simple Ncurses Hex Editor

![alt text](http)

## Usage
shexc (file)

| Key       | Action                                    |
|-----------|-------------------------------------------|
|  h,j,k,l  | Cursor Left, Down, Up, Right              |
|  0-9,A-F  | HEX Values                                |
|  s        | Save                                      |
|  q        | Quit                                      |

### Compile
gcc -lncurses shex.c -o shexc
